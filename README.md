## 📱 设计说明

### 主控制界面设计特点
- 深色 Navy 渐变背景 (#0d47a1 → #1565c0)
- 可自定义纹理层（木纹/星空等）
- 中央大圆形开关，带发光效果
- 实时功率显示（1250W 大字体）
- 底部4图标导航栏

### 定时界面设计特点
- 5种定时模式卡片式布局
- 倒计时、定时、循环、随机、点动
- 图标+标签+开关的清晰层级
- 玻璃拟态效果卡片

### 用电量界面设计特点
- 半圆形仪表盘可视化
- 4宫格关键数据展示
- 24小时用电曲线图
- 颜色编码：绿→黄→红（低→中→高功耗）

## 🎨 配色方案

### 推荐配色
```css
/* 深色主题背景 */
background: linear-gradient(135deg, #0f1419 0%, #1a1f2e 100%);

/* 开启状态强调色 */
accent-color: #0066cc;
glow-color: rgba(0, 102, 204, 0.3);

/* 文字颜色 */
text-primary: #FFFFFF;
text-secondary: #E0E0E0;
text-tertiary: #B0B0B0;

/* 状态颜色 */
status-good: #00e676;      /* 绿色 - 电量充足 */
status-warning: #ffa726;   /* 橙色 - 电量中等 */
status-danger: #ff5252;    /* 红色 - 电量不足 */
```
