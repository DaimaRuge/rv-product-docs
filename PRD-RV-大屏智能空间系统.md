# PRD：房车大屏智能空间系统

**版本**：V1.0  
**产品形态**：运行于 AI 主机 + 电视大屏显示  
**适配屏幕**：43 / 50 / 55 / 65 / 75 英寸  
**分辨率**：1920 × 1080  
**屏幕比例**：16:9  
**核心交互**：语音为主，蓝牙遥控为辅  
**产品定位**：面向房车旅居场景的智能空间操作系统

---

## 1. 项目背景

当前参考的 benchmark 产品，原本面向 10.1 英寸触控中控屏，交互方式以触控为主，产品形态更接近智能家居中控屏。

而本产品的运行环境、屏幕尺寸、交互模式、系统能力与价值重心均发生了明显变化：

- 运行在 43~75 英寸电视大屏上，而非近距离触控中控屏
- 电视主要承担显示器角色，系统实际运行于 AI 主机之上
- 交互方式从"触控操作为主"升级为"语音为主、遥控器为辅"
- 核心能力从"控制面板"升级为"语音 + Agent 的房车智能空间中枢"
- 产品价值从"设备控制"拓展到"空间控制 + 内容消费 + 导航出行 + 旅行服务"

因此，本产品不能简单沿用 benchmark 的页面组织方式，而需要形成适配大屏与语音产品形态的新架构。

---

## 2. 产品定位

### 2.1 产品定义
本产品是一套面向房车旅行生活方式的 **智能空间操作系统**，以"房车管家"为核心交互入口，通过语音、Agent 与大屏可视化能力，统一承接房车空间中的信息、设备、服务与内容。

### 2.2 角色定义：房车管家
房车管家不是单一语音按钮，而是整个系统的核心入口，承担三类中枢角色：

- **消息中枢**：承接系统通知、设备告警、状态提醒、旅行建议
- **控制中枢**：理解用户自然语言，调度设备与场景执行
- **服务中枢**：连接地图、音乐、视频、天气、旅行服务等能力

### 2.3 产品关键词
大屏、语音、Agent、房车空间、旅行生活方式、可视 + 可说、主动服务

---

## 3. 产品目标

### 3.1 业务目标
- 打造区别于传统 10.1 英寸中控屏的下一代房车大屏智能空间体验
- 建立"房车管家 + Agent"作为产品核心心智
- 强化房车场景下的大屏价值，包括导航、内容、氛围与旅行服务
- 为后续服务化、内容化、生态化拓展预留架构基础

### 3.2 用户目标
- 用户可以用自然语言快速理解和操作房车空间
- 用户可以在大屏上更轻松地查看状态、导航路线、视频内容与氛围信息
- 用户无需频繁进入复杂控制页，也能完成高频任务
- 用户感受到欢乐、阳光、旅行感的品牌氛围

### 3.3 体验目标
- **看得清**：远距可读，大屏可视化强
- **说得通**：语音交互自然，任务理解准确
- **控得住**：设备控制有反馈、有闭环、有安全逻辑
- **更主动**：系统能提醒、建议、协助，而不是被动等待点击
